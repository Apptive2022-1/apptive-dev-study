## 요구사항
- Jetpack Compose v1.2.1 사용
- 아래와 같은 패키지로 작성
```plain
|- (패키지)
    |- _enum // 필요한 경우
        - ...
    |- view
        - login.kt (원하는 경우 분리 가능)
    - MainActivity.kt
```
